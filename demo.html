GeoTIFFTileSource for OpenSeadragon

View local or remote TIFF images - without an image server!

Source code can be found at the project GitHub Repository. The GeoTIFFTileSource for OpenSeadragon depends on GeoTIFF.js.

Demo files (whole slide histology images - WSIs) can be downloaded from the OpenSlide project. Recommendation: download CMU-1.svs as a demonstration.

There are many TIFF formats, not all of which are GeoTIFF compatible. This applies to WSIs as well. For example, .ndpi files are not internally organized in a way that GeoTIFF.js can query efficiently, so slide viewing is very slow at best.
